# Linux Commands - Day 2

## File & Directory Commands

### mkdir
Create a directory
Example:
mkdir test

### rmdir
Remove empty directory
Example:
rmdir test

### touch
Create file
Example:
touch file.txt

### cp
Copy file
Example:
cp file.txt file2.txt

### mv
Move/rename file
Example:
mv file.txt newfile.txt

### rm
Delete file
Example:
rm file.txt

### tree
Show folder structure
Example:
tree

## Viewing Commands

### cat
Display file content
Example:
cat file.txt

### less
Scrollable view
Example:
less file.txt

### head
First 10 lines
Example:
head file.txt

### tail
Last 10 lines
Example:
tail file.txt

### tail -f
Live updates
Example:
tail -f file.txt

## Editors

### nano
Simple editor
Example:
nano file.txt

### vim
Advanced editor

Insert mode:
i

Save and exit:
:wq

Exit without saving:
:q!
