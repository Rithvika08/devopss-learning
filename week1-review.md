# Week 1 Review

## What I Learned

### Linux Basics
- Navigation (cd, ls, pwd)
- File operations (mkdir, touch, cp, mv, rm)

### Viewing Files
- cat, less, head, tail

### Editing Files
- nano, vim basics

### Permissions
- chmod (numeric & symbolic)
- chown, chgrp

### Users
- adduser, su, sudo

### Search & Filters
- grep, find, wc, sort, uniq, cut

### Piping
- Combining commands using |

### Process Management
- ps, top, htop, kill

### System Monitoring
- df, du, free

### Automation
- cron jobs

## Key Takeaways
- Linux is command-driven
- Small commands can be combined (pipes)
- Permissions are very important
- Automation saves time
